### OpenRouter integration sample

This example shows how to use the OpenRouter plugin to run a simple chat completion.

### Dependencies

The core stays lightweight; install plugin deps yourself.

```requirements
requests==2.32.3
```

Install with:

```commandline
pip install requests
```

### Environments

Set your OpenRouter API key:

```properties
OPENROUTER_API_KEY=sk-or-XXXXXXXXXXXXXXXXXXXXXXXX
```
